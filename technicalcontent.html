Technical Questions

1. What is the purpose of the HTTP protocol?

Correct Answer: b. To transfer hypertext documents
Explanation: HTTP is a protocol used for transmitting hypertext documents over the internet.

2. Which of the following languages is commonly used for web development on the client-side?

Correct Answer: c. JavaScript
Explanation: JavaScript is primarily used on the client-side for creating interactive web pages.

3. In databases, what does SQL stand for?

Correct Answer: b. Structured Query Language
Explanation: SQL is a standard language used to manage and query data in relational databases.

4. What is the primary purpose of CSS in web development?

Correct Answer: b. To style and layout web pages
Explanation: CSS (Cascading Style Sheets) is used to design and layout HTML content on web pages.

5. In programming, what does “API” stand for?

Correct Answer: b. Application Programming Interface
Explanation: API is a set of functions and procedures for interacting with external software components.

6. What does "DNS" stand for in networking?

Correct Answer: b. Domain Name System
Explanation: DNS translates domain names into IP addresses, allowing browsers to load internet resources.

7. What is the function of the GET method in HTTP?

Correct Answer: b. To retrieve data from the server
Explanation: The GET method requests data from a specified resource on a server.

8. Which of the following is a NoSQL database?

Correct Answer: c. MongoDB
Explanation: MongoDB is a NoSQL database that stores data in JSON-like documents.

9. In CSS, which property is used to change the text color of an element?

Correct Answer: b. color
Explanation: The CSS color property changes the text color of an element.

10. Which data structure follows the Last In First Out (LIFO) principle?

Correct Answer: b. Stack
Explanation: A stack follows the LIFO (Last In First Out) principle, meaning the last element added is the first to be removed.

11. Which programming language is known as the foundation of web development for creating web pages?

Correct Answer: b. HTML
Explanation: HTML (HyperText Markup Language) is the standard markup language for creating web pages.

12. What does "IP" stand for in networking?

Correct Answer: a. Internet Protocol
Explanation: IP, or Internet Protocol, is responsible for addressing and routing data packets across networks.

13. What is the main purpose of a firewall in network security?

Correct Answer: b. To protect against unauthorized access
Explanation: Firewalls filter network traffic to prevent unauthorized access to or from a private network.

14. Which of the following is an example of a relational database?

Correct Answer: c. MySQL
Explanation: MySQL is a popular relational database management system that organizes data into tables.

15. In JavaScript, which keyword is used to declare a constant variable?

Correct Answer: c. const
Explanation: In JavaScript, const is used to declare variables whose values cannot be changed once assigned.
